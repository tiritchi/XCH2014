La Xtrem Cergy Hunting est le premier street war se déroulant dans Cergy et réunissant l’ensemble des écoles post bac de la ville. L’inscription au jeu est donc réservée aux étudiants de la ville et dont le domicile se trouve sur cette carte. Il s’agit d’un jeu, nous comptons sur votre fair play, et sur votre respect des règles. Nous avons mi à votre disposition un système de mailling pour que vous nous remontiez tout problèmes survenant durant le jeu. Le jeu est construit pour pouvoir jouer un maximum de temps, mais aussi pour récompenser les personnes les plus douées !


Contexte du Jeu

Vous êtes recruté par un responsable de la mafia locale. Votre job est d’arroser des cibles dans la ville. Pour cela il va falloir mener une enquête pour retrouver votre cible, et l’arroser !


Inscription

L’inscription se déroule en deux phases :
Tout d’abord une inscription sur notre plateforme internet. Vous devez impérativement fournir des informations exactes et le plus précises possible. Sinon on ne pourra pas vous retrouver, et vous ne pourrez pas retrouver votre cible. Vous aurez besoin d’enregistrer une photo de vous sur notre plateforme. Cette photo apparaitra sur votre fausse carte d’identité ainsi que sur les contrats que se verront remettre les personnes qui vous chercheront. Attention au format de la photo, elle doit être de la taille d’une photo d’identité avec les mêmes spécifications. (visage visible, photo claire, moins de 3 mois). Vous aurez 5 coordonnées à remplir. Elles représentent les endroits où vous avez l’habitude de vous rendre. Vous devez impérativement mettre les coordonnées des endroits où vous vivez, ou votre école est, et par exemple là ou vous vous promenez, faites vos courses etc …
Merci d’utiliser une boite mail que vous utilisez régulièrement. Ainsi qu’un numéro de téléphone valable, il pourra être utilisé pour vous contacter. Une fois cette inscription faite, vous serez contacté par mail pour la deuxième phase de recrutement. La deuxième phase de recrutement est cette fois un rendez-vous physique avec votre contact dans la mafia, ambiance des années 30 au programme ... Votre agent de liaison dans le milieu de la pègre vous contactera pour vous fixer un rendez-vous. Il vous y sera remis votre pistolet à eau et votre code de joueur. Un rappel des règles vous sera fait et vous pourrez poser toutes vos questions. C’est aussi le moment où vous payerai votre inscription au jeu.



Le code joueur

Lors de votre inscription vous recevrez automatiquement un code joueur dans votre espace personnel. Que ce soit noté sur un bout de papier ou enregistré dans votre téléphone, vous devez en permanence avoir ce code sur vous.
Ce code joueur est très important et ne doit être communiquer à personne, il sert à vous identifier. L'unique moment où vous aurez à le communiquer c'est à la personne qui vous à arrosé.
Le code joueur doit alors être entré dans votre espace personnel à l'endroit indiqué dans le contrat que vous avez rempli.


Le jeu

Nous vous proposons un évènement réparti en 3 manches.

1ere manche :
Chaque participant reçoit 5 contrats. Il a une semaine pour honorer un maximum de contrats. La première manche se finira par un classement des meilleurs arroseurs.

2ème manche :
Vous recevez un contrat et vous devez l’honorer le plus rapidement possible. Si vous réussissez à abattre votre cible, vous recevrez son contrat par message. Si vous vous faite abattre, vous êtes éliminé de cette phase de jeu et votre cible est transférée à celui qui vous a abattu. Cette phase s’arrête au bout de deux semaines. Les survivants sont automatiquement sélectionnés pour la 3ème manche.

3ème manche :
Manche fait office de combat final. Les concurrents ont une après-midi pour arroser le boss de la mafia. Le boss traversera la ville de Cergy et son itinéraire vous sera transmis. Vous devrez alors l’intercepter et l’arroser. L’ensemble des candidats éliminé seront alors embauché pour le défendre et arroser ses assaillants. Préparer vous a une finale épique ! Le premier qui réussira à arroser le boss remportera le jeu.

Validation de l’élimination d’un joueur :
Chaque joueur doit porter en permanence sa fausse carte d’identité. Sur cette carte d’identité est inscrit un numéro de joueur. Lorsque vous arrosez votre cible, elle est tenu de vous présenter sa carte de joueur et vous devez noter son code joueur. Il ne vous reste plus qu’à entrer ce code sur notre plateforme internet dans le contrat associé. Vous recevrez automatiquement un nouveau contrat.
